About Us

About the Application

The User Management System is a simple web application that allows you to manage users. It provides functionality to add new users, view existing users, and delete users.

This application was built using plain HTML, CSS, and JavaScript for the frontend and Node.js with Express.js for the backend. User data is stored in a JSON file.

About the Developer

This application was created as a demonstration of simple web development skills including:

  • HTML, CSS, and JavaScript
  • Node.js and Express.js
  • RESTful API design
  • Middleware implementation
  • File-based data storage
